Nerine OS

A Debian-based gaming and customization-focused Linux distribution.

Download Nerine OS Explore Features

Built for Gaming. Designed for Customization. Powered by Debian.

Nerine OS provides a modern, fast, and stable Linux experience with full compatibility with the Debian ecosystem. Crafted for gamers, creators, and power users, it offers optimized desktop environments, performance enhancements, and a unified customization layer.

With installation profiles for Gaming, Normal, and Minimal setups, Nerine OS adapts to any system and use case.

Core Highlights

Gaming Mode

Includes Steam, Heroic, Lutris, Wine Staging, ProtonUp-Qt, MangoHUD, Gamescope, and Vulkan optimizations.

Three Desktop Environments

KDE Plasma, XFCE, and GNOME editions — each fully themed, configured, and optimized.

NerineBoost Engine

System service that automatically tunes CPU, GPU, I/O, audio latency, and compositor effects during gameplay.

Unified Customization

Switch global themes, icons, layouts, and boot visuals using the NerineCustomizer tool.

Debian Compatibility

Full support for APT, Debian Testing repositories, and all standard `.deb` packages.

Installation Profiles

Choose between Gaming, Normal, or Minimal installation modes through a custom Calamares installer.
