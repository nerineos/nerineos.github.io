What is Nerine OS?

Nerine OS is a modern, performance-focused Linux distribution built on top of Debian Testing. It introduces advanced gaming optimizations, updated graphics stacks, custom system services, and a polished interface across KDE Plasma, GNOME, and XFCE.

The goal is to provide an operating system that is fast, stable, customizable, and suitable for gaming, creative work, and everyday use while maintaining full compatibility with APT and the Debian ecosystem.

Core Features

Modern Debian Base

Powered by Debian Testing for modern, stable, and secure software packages while still maintaining the familiar APT package manager.

Gaming-Optimized Mode

Steam, Heroic, Lutris, Gamescope, Vulkan, MangoHUD, Wine Staging, and Proton tools included with automatic system tuning.

NerineBoost Engine

A system service that enhances performance automatically during gameplay by adjusting CPU governors, GPU profiles, audio latency, I/O schedulers, and compositor settings.

Advanced Customization Tools

Change global themes, layouts, icons, accents, bootsplash screens, login screens, and desktop effects using the NerineCustomizer.

Calamares Installer

A modern graphical installer with selectable installation modes: Gaming, Normal, and Minimal, plus driver customization and filesystem options.

Updated Graphics Stack

Includes updated Vulkan, Mesa, DXVK, and vkBasalt for smooth gaming performance, along with optional proprietary NVIDIA drivers.

Performance Kernels

Support for performance-tuned kernels such as Liquorix and XanMod, ideal for gaming and low-latency workloads.

Optimized Desktop Environments

KDE Plasma, GNOME, and XFCE are themed, stabilized, and configured for consistent usability and efficient performance.

Installation Profiles

Nerine OS includes three different installation modes to fit your system and use case.

Gaming Installation

Installs the full gaming stack including Steam, Heroic, Lutris, ProtonUp-Qt, Wine Staging, Gamescope, MangoHUD, and NerineBoost.

Normal Installation

Includes desktop apps, office suite, browser, customization tools, and system utilities.

Minimal Installation

A lightweight base with only the essentials, allowing advanced users to build the system their way.

Performance Highlights

Fast Boot Times

Optimized system services allow Nerine OS to boot significantly faster than standard Debian.

Low Latency Audio

PipeWire configuration reduces audio delay and improves microphone quality during gaming.

Stable Frame Times

NerineBoost adjusts system resources dynamically to reduce frame drops and micro-stutter.

Updated Hardware Support

Newer kernels, firmware, and graphics drivers offer improved compatibility with modern GPUs.
